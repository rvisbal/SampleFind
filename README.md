# SampleFind

## Overview
SampleFind is a Windows .NET application that allows users to load text or log files and create multiple filters to display only the lines that match specific conditions.

## Features
- Load and analyze text files (.txt) and log files
- Create and apply multiple custom filters simultaneously
- Show only the lines that match your specified conditions
- Easy-to-use interface for Windows users
- Quick filtering for large log files

## Installation
```bash
# Download the latest release
# Run the installer and follow the on-screen instructions
```

## Usage
```
1. Launch SampleFind
2. Open a text or log file using File > Open or drag and drop
3. Create filters by specifying your search conditions
4. Apply filters to see matching lines
```

## Configuration
Filters can be configured with various matching conditions including:
- Text contains/does not contain
- Regular expressions
- Case sensitivity options

## Requirements
- Windows operating system
- .NET Framework (version details)
- Minimum screen resolution: 1024x768
